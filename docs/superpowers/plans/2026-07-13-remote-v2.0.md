# startup-law-mcp v2.0 원격 배포 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fly.io에 streamable-http 원격 MCP 서버 배포 — URL 하나로 설치 없이 연결, main push 자동 재배포.

**Architecture:** `register_tools`에 `include_admin` 게이트 추가(원격 12개/로컬 13개), 얇은 `server_http.py` 엔트리포인트, 인덱스를 이미지에 굽는 Dockerfile, auto-stop fly.toml, 경로 필터 fly-deploy 워크플로. 원격 서버는 시크릿 0개(읽기 전용).

**Tech Stack:** FastMCP streamable-http, Docker(python:3.12-slim), Fly.io, superfly/flyctl-actions.

**스펙:** [docs/superpowers/specs/2026-07-13-remote-v2.0-design.md](../specs/2026-07-13-remote-v2.0-design.md)

## Global Constraints

- 원격 표면은 정확히 12개 도구 — `sync_programs`만 제외. 로컬 stdio는 13개 불변.
- server_http.py는 얇게(신규 로직 금지 — server.py의 SERVER_INSTRUCTIONS·register_tools 재사용).
- 원격 서버에 API 키·시크릿 주입 금지 (데이터는 이미지에 구움).
- fly-deploy.yml은 경로 필터(`**.py`, `data/**`, `Dockerfile`, `fly.toml`, 자기 자신) + concurrency 그룹.
- **병합 순서**: feat/v2.0은 T3(FLY_API_TOKEN 등록·첫 배포 성공)까지 마친 뒤 main 병합 — 토큰 없는 상태의 실패 런 방지.
- conventional commits. `python -m pytest` 전체 통과 유지 (기준 126).

① `server.py` — 시그니처 교체:
```python
def register_tools(mcp: FastMCP, include_admin: bool = True) -> None:
    """도구 등록 단일 지점 — 로컬 stdio(13개)와 원격 HTTP(12개)가 공유.

    include_admin=False면 관리 도구(sync_programs)를 제외한다 — 원격 공개판에서
    아무나 서버측 API 수집을 유발하지 못하게 (데이터 갱신은 재배포로).
    """
```
② `sync_programs` 도구 정의 전체를 함수 **마지막**으로 옮기고 다음으로 감싼다:
```python
    if not include_admin:
        return

    @mcp.tool()
    def sync_programs() -> dict:
        ...  # 기존 본문 그대로 (들여쓰기 한 단계 유지)
```
(기존 module-level `register_tools(mcp)` 호출은 기본값 True라 변경 불필요 — 단 명시성을 위해 `register_tools(mcp, include_admin=True)`로 교체.)

③ `server_http.py` 생성:
```python
"""원격 HTTP 엔트리포인트 (Fly.io) — 읽기 전용 12개 도구.

sync_programs(관리)는 원격 공개판에서 제외한다. 데이터 갱신은 주간 동기화
PR 머지 → main push 자동 재배포(fly-deploy.yml)로 이뤄진다.

로컬 확인: python server_http.py  →  http://127.0.0.1:8080/mcp
검증:      python scripts/remote_smoke.py http://127.0.0.1:8080/mcp
"""
from __future__ import annotations

from mcp.server.fastmcp import FastMCP

from server import SERVER_INSTRUCTIONS, register_tools

mcp = FastMCP("startup-law", instructions=SERVER_INSTRUCTIONS,
              host="0.0.0.0", port=8080, stateless_http=True)
register_tools(mcp, include_admin=False)


if __name__ == "__main__":
    mcp.run(transport="streamable-http")
```
**주의:** FastMCP 생성자 kwargs(host/port/stateless_http)는 settings로 전달된다. 사용 중인 mcp 버전에서 kwargs가 거부되면(`TypeError`) `mcp.settings.host = "0.0.0.0"` 식 대입으로 대체하고 리포트에 기록하라 — 동작 계약(0.0.0.0:8080, stateless, /mcp)은 불변.

`scripts/remote_smoke.py`:
```python
"""원격 MCP 서버 스모크 — initialize → tools/list(12개) → search_law 실호출.

사용:  python scripts/remote_smoke.py http://127.0.0.1:8080/mcp
       python scripts/remote_smoke.py https://<앱>.fly.dev/mcp
"""
from __future__ import annotations

import asyncio
import sys

from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client


async def main(url: str) -> None:
    async with streamablehttp_client(url) as (read, write, _):
        async with ClientSession(read, write) as session:
            await session.initialize()
            tools = await session.list_tools()
            names = sorted(t.name for t in tools.tools)
            print(f"tools({len(names)}):", ", ".join(names))
            assert len(names) == 12, "원격 표면은 12개여야 함"
            assert "sync_programs" not in names
            result = await session.call_tool(
                "search_law", {"query": "창업 세액감면", "limit": 3})
            text = str(result.content)[:300]
            print("search_law 응답 발췌:", text)
            assert "조세특례제한법" in text
    print("SMOKE OK")


if __name__ == "__main__":
    asyncio.run(main(sys.argv[1]))
```
